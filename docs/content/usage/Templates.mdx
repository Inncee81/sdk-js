## Introduction

A workflow can be of be a type of instance or template. A workflow without a business defined is referred to as a template (link)

  dictionary Workflow : Template {
                required Template template;
                required ByteString name;
                boolean is_template = true;
                required User creater;
                required PartnershipConnection connection;
                DocumentGroup document;
                required Business owner;
            };

## Usage

### list

Retrieve a list of Templates.

#### Usage

``` javascript
const templates = await templateService.list();
```

#### Response

``` json

[
    {
        "_id": "af6d8dfc-88b5-41f6-bbf5-825130390d64",
```

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| name | string | <CheckIcon color="green.5"/> |  'template name (order)' |
| is_template | boolean | <CheckIcon color="green.5"/> | true |
| creator_id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| connection_id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| document_group_id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| owner_id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
