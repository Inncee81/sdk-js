## Introduction

The transitions api gives the ability to fully configure each transition between the stages (link) of a work flow template. (link)

## Usage

### list

retreive the list of transitions.

#### Usage 

``` javascript
const transitions = await transitionsService.list();
```

#### Response

``` json
{

}
```

### get

retrieve a transition by Id.

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| id | Xuid | <CheckIcon color="green.5"/> | '5076455a-a18e-11ea-8618-0242ac120002' |

``` javascript
const transition = await transitionsService.get('5076455a-a18e-11ea-8618-0242ac120002');
```
